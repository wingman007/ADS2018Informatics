Един метод се нарича устойчив, ако относителният ред на елементите с равни ключове остава непроменен
в процеса на сортиране. Устойчивите методи на сортиране се предпочитат тогава, когато елементите
на множеството вече са сортирани съгласно някакъв друг критерий. Например, нека искаме
да сортираме файловата си система по име и по разширение: файловете се нареждат по име, като
тези с еднакво име се нареждат по разширение. Това може да стане на две стъпки: 1) сортиране по
разширение, и 2) сортиране по име. Докато при първото сортиране (по разширение) устойчивостта
на алгоритъма е без значение, при повторното сортиране (по име) използването на устойчив метод
вече ще бъде задължително.


*сортиране чрез броене
